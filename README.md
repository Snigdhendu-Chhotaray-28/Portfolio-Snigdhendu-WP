# Portfolio-Snigdhendu-WP
A modern and interactive portfolio designed to showcase my projects, experience, and GitHub activity. Built with a sleek dark UI, smooth animations, and responsive design, it highlights my frontend development skills while providing an engaging user experience across all devices. Tech stack includes HTML, CSS, JavaScript, React, and Tailwind CSS.

<h1>🌌 Developer Portfolio Website</h1>
The Developer Portfolio Website is a modern, visually immersive personal platform designed to showcase projects, experience, and technical skills in an engaging and interactive way. Built with a dark space-inspired aesthetic and smooth UI interactions, the portfolio highlights professional work, development journey, and GitHub activity while maintaining high performance and responsiveness.

It acts as a digital identity for a frontend developer, presenting projects, technical expertise, and real-world experience through clean design and thoughtful user experience.
